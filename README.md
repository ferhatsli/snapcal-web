# SnapCal Web

Marketing website for SnapCal, implemented with Next.js App Router and Tailwind CSS.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4

## Routes

- `/` Home landing page
- `/pricing` Plan comparison and subscription CTA
- `/faq` Frequently asked questions
- `/privacy` Privacy policy
- `/terms` Terms of service

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run lint
npm run build
npm run start
```
